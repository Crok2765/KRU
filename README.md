# KRU - Kader's Raid Utilities

## 📖 Descripción

Kader's Raid Utilities (KRU) es una colección de herramientas avanzadas para World of Warcraft 3.3.5a diseñada para líderes de banda, asistentes y oficiales que necesitan información precisa en tiempo real durante encuentros de raid.

KRU proporciona módulos especializados para el seguimiento de habilidades, monitoreo de jugadores, estadísticas de combate, gestión de invitaciones y otras utilidades esenciales para la coordinación eficiente de grupos y bandas.

> [!NOTE]
> Compatible con World of Warcraft 3.3.5a.

> [!TIP]
> Diseñado para proporcionar información clara y útil durante encuentros de raid sin sobrecargar la interfaz.

---

## ✨ Características

* ✔ Monitoreo en tiempo real.
* ✔ Herramientas para líderes de banda.
* ✔ Sistema modular.
* ✔ Configuración avanzada.
* ✔ Marcos movibles y personalizables.
* ✔ Reportes rápidos.
* ✔ Compatible con grupos y bandas.
* ✔ Integración con múltiples módulos especializados.

---

## 📦 Módulos Incluidos

### ⚔️ Raid Utilities

* Aura Monitor
* Cooldowns Tracker
* Combat Time
* Mana Monitor
* Loot Utilities
* Raid Invites

### 🎯 Seguimiento de Habilidades

* Secretos del Oficio (Tricks of the Trade)
* Redirección (Misdirection)
* Sunder Tracker

### 📊 Información y Estadísticas

* Monitoreo de jugadores
* Reportes rápidos
* Seguimiento de eventos de raid

### ⚙️ Configuración

* Escala de marcos
* Tamaño de fuente
* Frecuencia de actualización
* Bloqueo y desbloqueo de ventanas
* Posicionamiento personalizado
